## Prerequisites

Before installing Reysin, make sure you have the following installed on your system:

- Node.js (version 21.0.0 or higher)
- pnpm (`npm install -g pnpm`)

<Aside type={'caution'}>
  You can also use npm or other packages manager but we recommend pnpm
</Aside>

## Installation

1. Open your terminal.

2. Create a new Reysin project using the following command:

   ```bash
   npm create reysin-app@latest my-reysin-project
   ```

3. Follow the on-screen instructions to configure your project.

4. Once the installation is complete, navigate to your project directory:

   ```bash
   cd my-reysin-project
   ```

5. Start the development server:

   ```bash
   pnpm dev
   ```

Your Reysin application is now running! Open your browser and go to `http://localhost:3000` to see your new application.
