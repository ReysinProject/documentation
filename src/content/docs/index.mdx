## Get started in a flash

```bash
npm create reysin-app@latest my-reysin-project
cd my-reysin-project
npm run dev
```

## Why Reysin?

- **Based on Vite**: Ultra-fast development and instant reloading
- **MVVM Architecture**: Clear separation between logic and presentation
- **Powerful CLI**: Generate components, services, and ViewModels effortlessly
- **Built-in IoC Container**: Simplified dependency management
- **100% TypeScript**: Benefit from strong typing for robust code
- **Extensible**: Adapt Reysin to your specific needs
